# تكامل Shopify Integration

## نظرة عامة
صفحة تكامل Shopify تتيح للمستخدمين ربط متاجر Shopify الخاصة بهم مع نظام Saler لإدارة المبيعات والعملاء بشكل مركزي.

## المميزات الرئيسية

### 1. إعداد الاتصال (Connection Setup)
- إدخال رابط متجر Shopify
- إدارة مفاتيح API
- تكوين إعدادات التطبيق الخاص
- اختيار الأذونات المطلوبة
- اختبار الاتصال

### 2. إعداد Webhooks
- تكوين إشعارات فورية للطلبات والعملاء والمنتجات
- إدارة روابط Webhooks
- إنشاء مفاتيح التحقق
- اختبار تسليم Webhooks
- أمثلة كود للمطورين

### 3. ربط البيانات (Data Mapping)
- ربط حقول عملاء Shopify مع نظام Saler
- ربط بيانات الطلبات
- ربط بيانات المنتجات
- دعم الحقول المخصصة

### 4. إعدادات المزامنة
- المزامنة التلقائية أو اليدوية
- تكرار المزامنة (فوري، كل ساعة، يومي، أسبوعي)
- حل التعارضات
- إدارة استيراد/تصدير البيانات

### 5. لوحة المراقبة
- حالة الاتصال في الوقت الفعلي
- نشاط المزامنة والأخطاء
- إحصائيات الأداء
- سجل الأخطاء والتحذيرات

## التقنيات المستخدمة
- **React** مع TypeScript
- **Tailwind CSS** للتصميم
- **Lucide React** للأيقونات
- **Next.js App Router**

## الوظائف

### الخطوات التفاعلية
1. **إعداد الاتصال** - توصيل متجر Shopify
2. **إعداد Webhooks** - تكوين الإشعارات
3. **ربط البيانات** - ربط حقول البيانات
4. **إعدادات المزامنة** - تكوين المزامنة
5. **المراقبة** - مراقبة الأداء

### الحالة (State Management)
- `config` - إعدادات التكامل
- `currentStep` - الخطوة الحالية
- `connectionStatus` - حالة الاتصال
- `syncStatus` - حالة المزامنة
- `showCodeExample` - إظهار أمثلة الكود

### الوظائف المساعدة
- `handleConnect` - معالجة الاتصال
- `copyToClipboard` - نسخ إلى الحافظة
- `renderStepIndicator` - عرض مؤشر الخطوات
- `renderConnectionSetup` - عرض إعداد الاتصال
- `renderWebhookConfig` - عرض إعداد Webhooks
- `renderDataMapping` - عرض ربط البيانات
- `renderSyncSettings` - عرض إعدادات المزامنة
- `renderMonitoringDashboard` - عرض لوحة المراقبة

## المتطلبات

### من جانب Shopify
- حساب Shopify متجر نشط
- صلاحيات المدير
- HTTPS مفعل
- دعم webhooks

### من جانب النظام
- اتصال مستقر بالإنترنت
- خادم يدعم webhooks
- دعم SSL/TLS

## الأمان
- تشفير مفاتيح API
- التحقق من صحة Webhooks
- أذونات محدودة حسب الحاجة
- تسجيل العمليات

1. انتقل إلى `/dashboard/integrations/shopify`
2. اتبع الخطوات المفصلة
3. اختبر الاتصال
4. راقب الأداء
